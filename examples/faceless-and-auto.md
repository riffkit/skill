# Example — run a faceless account on Auto

**You have:** a faceless account to feed, and no on-camera talent.
**You want:** a steady stream of videos without filming anyone.

Leave the character out — Riffkit's **Auto** mode generates the on-camera person for you (no avatar to set up).

```
riff this trending video into mine, auto character
```

What you get is a post-ready video — voiceover, timed captions, cover, hashtags — walking the same emotional arc as the source, with no footage re-uploaded.

https://github.com/user-attachments/assets/b7e6ea34-9f9a-4d3c-a191-cabd84ca6f00

Want the *same* character across every post instead? Set up a persona once and reuse it. Run it from your agent or at [riffkit.ai](https://riffkit.ai).
